# vue-githun-explorer

## 功能
- 输入指定的username/repository的关键字，可检索到对应的repo的目录列表
- 点击列表的中目录的name，可以跳到点击的下层目录
- 点击Back按钮可以返回上层目录

## Project setup
```
yarn install
```
